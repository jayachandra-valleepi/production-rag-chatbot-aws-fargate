📊 RAG Stock Market Basics Chatbot
🚀 Overview

This project is a Retrieval-Augmented Generation (RAG) chatbot that answers stock market basics questions using a custom knowledge base.

Instead of relying only on a language model, it:

Retrieves relevant information from documents
Uses an LLM to generate accurate, context-based answers
🎯 Goal

To build a chatbot that can explain stock market concepts like:

What is a stock?
What is NSE/BSE?
What is a mutual fund?
Basic trading concepts
🧠 How It Works (Simple Flow)
User asks a question
System searches relevant content from stored documents
Retrieved data is sent to the LLM
LLM generates a clear answer

👉 In short:
Search + AI = Better Answers
